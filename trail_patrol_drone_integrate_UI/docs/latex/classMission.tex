\doxysubsection{Detailed Description}
Main mission management class for autonomous drone navigation. 

This class handles\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{structWaypoint}{Waypoint}} queue management (initial mission + custom waypoints)
\item \mbox{\hyperlink{classMission}{Mission}} state control (start, pause, resume, stop)
\item Obstacle avoidance coordination with navigation node
\item UI status updates and progress tracking
\item Fly-\/over and fly-\/around avoidance protocols
\end{DoxyItemize}

The mission publishes waypoints to the navigation node and monitors waypoint completion, adjusting for obstacles as needed.

Activates fly-\/over obstacle avoidance protocol. 

Commands the drone to climb vertically to 7.\+0m altitude, then traverse over the obstacle to the original waypoint position, resuming mission from there. Used when no clear lateral path exists or for emergency close obstacles.

Two-\/phase process\+:
\begin{DoxyEnumerate}
\item FLY\+\_\+\+OVER\+\_\+\+UP\+: Climb to 7.\+0m at current position
\item FLY\+\_\+\+OVER\+\_\+\+ACROSS\+: Move to interrupted waypoint at 7.\+0m, then descend
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
Avoidance mode remains false during fly-\/over

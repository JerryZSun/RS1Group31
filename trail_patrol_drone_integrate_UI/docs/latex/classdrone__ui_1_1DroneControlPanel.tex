\doxysubsection{Detailed Description}
RViz2 panel widget for comprehensive drone control and monitoring. 

This class implements a Qt-\/based control panel that integrates with RViz2 to provide a graphical interface for drone mission management. It supports multiple control modes\+:

{\bfseries{\mbox{\hyperlink{classMission}{Mission}} Mode\+:}}
\begin{DoxyItemize}
\item Start/pause/resume/stop predefined missions
\item Add custom waypoints to mission queue
\item Clear custom waypoints while preserving mission path
\item Monitor mission progress and waypoint completion
\end{DoxyItemize}

{\bfseries{Manual Mode\+:}}
\begin{DoxyItemize}
\item Direct velocity control with arrow buttons
\item Altitude control with up/down commands
\item Rotation control with left/right commands
\item Emergency stop for immediate halt
\end{DoxyItemize}

{\bfseries{Monitoring\+:}}
\begin{DoxyItemize}
\item Real-\/time position display (X, Y, Z)
\item Current altitude indication
\item Drone speed calculation and display
\item \mbox{\hyperlink{classMission}{Mission}} state and progress tracking
\item Status messages with timestamps
\end{DoxyItemize}

The panel publishes commands to mission and navigation nodes via ROS2 topics and subscribes to telemetry and status updates for real-\/time display.

\begin{DoxyNote}{Note}
This panel requires \mbox{\hyperlink{mission_8cpp}{mission.\+cpp}} and \mbox{\hyperlink{navigation_8cpp}{navigation.\+cpp}} nodes to be running 
\end{DoxyNote}

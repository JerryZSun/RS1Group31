\doxysubsection{Detailed Description}
RViz2 panel for drone control and mission management. 

This header defines the Drone\+Control\+Panel class which provides a Qt-\/based graphical user interface for controlling autonomous drone missions within RViz2.

Features\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{classMission}{Mission}} control (start, pause, resume, emergency stop)
\item Custom waypoint queue management
\item \mbox{\hyperlink{structWaypoint}{Waypoint}} library for click-\/to-\/go navigation
\item Manual control mode with directional buttons
\item Real-\/time telemetry display (position, altitude, speed)
\item \mbox{\hyperlink{classMission}{Mission}} progress and status monitoring
\end{DoxyItemize}
